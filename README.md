## Development

Clone the repo:

```bash
git clone https://github.com/soywod/soywod.git
```

Install dependencies:

```bash
yarn install
```

Start the Next.js development server:

```bash
yarn start
```

To build a static version of the blog:

```bash
yarn build
```
